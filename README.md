## 🔬 研究基础介绍 (Research Summary)
    主要从事非完整信息下电力系统状态感知、复杂环境下电能质量监测与治理、新型电力系统优化与控制等方向的研究与开发工作。主持国家自然科学基金青年基金项目、上海市白玉兰人才计划浦江项目D类、国家电网等科研项目多项。在智能电网、计算机、控制等领域顶级期刊发表论文20余篇，以第一作者出版英文专著1部。

## 💡 主持科研项目 (Research Projects)
### 1. 国家自然科学基金青年科学基金项目 (62303301)
* **项目名称：** 面向新型电力系统电源侧的测量离群值影响下分布式状态估计
* **起止年月：** 2024.01 - 2026.12
* **简介：** 研究测量离群值影响下分布式状态估计问题。建立测量离群值数学模型，构建主动式检测与处理机制，并将其嵌入状态估计框架中，实现估计器参数设计与优化。

### 2. 上海市白玉兰人才计划浦江项目 (23PJD040)
* **项目名称：** 新型电力系统环境下基于数据驱动的多能互补发电态势感知方法研究
* **起止年月：** 2023.10 - 2024.09
* **简介：** 针对新能源随机性与波动性带来的复杂耦合，建立数学模型并构建态势感知算法，解决发电机事件辨识及高可信度功率预测问题。

### 3. 国家电网上海电力公司项目
* **项目名称：** 基于数据驱动的工商业负荷用电特性分析及可调潜力评估研究
* **起止年月：** 2024.01 - 2024.12
* **简介：** 构建 Transformer 神经网络模型提取负荷时序特征，利用随机森林揭示电力与非电力因素关系，并基于卷积神经网络预测储能型负荷的可信可调区间。
