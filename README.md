## 🔬 简要介绍 
主要从事非完整信息下电力系统状态感知、复杂环境下电能质量监测与治理、新型电力系统优化与控制等方向的研究与开发工作。主持国家自然科学基金青年基金项目、上海市白玉兰人才计划浦江项目D类、国家电网等科研项目多项。在智能电网、计算机、控制等领域顶级期刊发表论文20余篇，以第一作者出版英文专著1部。
